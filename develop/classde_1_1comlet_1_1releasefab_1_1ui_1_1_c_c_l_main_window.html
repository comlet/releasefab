Detailed Description

The main menu and ToolBar are defined in this class.

The rest of the window consists of a TabFolder. The TabFolder itself contains two different kinds of tabs. One represents the information of every component in each delivery. The other one is responsible for the strategy used to receive the information. The first column always shows the component tree.
