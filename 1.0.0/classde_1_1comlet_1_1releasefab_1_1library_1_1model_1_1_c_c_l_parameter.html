Detailed Description

An array of CCLParameter serves as input for the abstract getData method of ACLAssignmentStrategy.

Using a dedicated data type allows for more flexibility and future expansion. It is important to document the whole purpose of the parameter in mInfo to help the developer implement the getData() method. The current type of mValue is String, later on it might be replaced by a generic type.
