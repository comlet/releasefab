Detailed Description

A delivery is a snapshot of information about components at a certain time.

A delivery has a unique name. This name is used as a key in CCLComponent. This class contains the global delivery data - public properties. These properties are the same for every component taking part in the delivery. The additional information about the delivery is saved in the component itself and can be accessed by the name of the delivery. The number of additional information in a component is dependent on the number of classes derived from ACLDeliveryInformation.
