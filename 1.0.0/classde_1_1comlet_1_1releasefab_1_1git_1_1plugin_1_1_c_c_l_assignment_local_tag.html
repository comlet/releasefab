Detailed Description

Assigns the local Git TAG.

Therefore the HEAD of the local repository must be synched to this TAG. The location of the repository is either configured or the project root is used. This is an external AssignmentStrategy. It can be registered to another plugin using the method addToExternalPlugin(String).
